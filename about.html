Uni-Buddy



Uni-Buddy or University Buddy is a one stop solution for all university going students. Uni-Buddy is a Student Information and support System which covers various aspects of managing students records and academic information.

Students as well as faculty can log in to their accounts and view their information and access various features like CGPA calculator, attendance, exam marks, scientific calculator, time table, holidays, list of courses, faculty list, etc.

Students face many problems of different platforms. They need Microsoft teams for online classes, E-lab for coding, academia for time table, attendance, faculty list, course plan and important notices. Uni-Buddy is a one-stop solution for university going students. They can find all the things like CGPA calculator, time table, attendance, course plan, event registrations, online classes, notes, tests and quizzes all at one place.

Programming Language: Java
The entire project is written in Java, which is a versatile, object-oriented programming language.

Standard Libraries: Java Standard Library
The project utilises the standard libraries that come with Java for various functionalities like input/output, data storage etc.

IDE (Integrated Development Environment)
The code can be developed using any Java-supporting IDE like Eclipse, IntelliJ IDEA, or NetBeans.

Console Interaction
The project interacts with users through the command-line console using the `java.util.Scanner` class.

Data Storage
The project uses in-memory data structures like lists and maps to store information such as student details, faculty details, attendance, and more. For a real-world application, you might consider using a database (such as MySQL) to store persistent data.

OOP Principles
Object-Oriented Programming (OOP) principles are followed throughout the codebase, using classes and objects to model different entities and their behaviours.
